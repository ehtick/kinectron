Kinectron Getting Started Example

Instructions:

This example demonstrates how to connect to a Kinectron server, initialize the Kinect, and display color and depth feeds.

When you load this page, it will:

  1. Connect to the Kinectron server (default: localhost)
  2. Set the Kinect type to Azure
  3. Initialize the Kinect hardware

Once connected and initialized, you can:

  • Press ENTER to start the color feed
  • Press UP ARROW to start the depth feed
  • Press RIGHT ARROW to stop all feeds

Note: Make sure your Kinectron server is running before testing this example.
